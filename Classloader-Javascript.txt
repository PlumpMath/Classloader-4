Ideas

    - replace slashes with strings
  - Class reference verification plugin
  - If you can parse Javascript, you can show methods on classes

    - You'll need a settings file, to denote the main class
  - A command for writing a testcase script for the current class
  - refactor command
    - rewrite the package statement, class name, and any references to it in other files to the files current class name and package location.
  - snippet command
    - select some text and create a snippet out of it.
